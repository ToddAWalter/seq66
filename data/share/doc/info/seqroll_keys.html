Pattern Piano-roll Keystrokes Some of the Tools operations have keystroke support. Other keys are like the "Common" keystrokes, but apply to selected notes.
Copy/Cut/Paste/Undo/Redo Ctrl-C, Ctrl-X, Ctrl-V, Ctrl-Z, Shift-Ctrl-Z
Select all notes Ctrl-A
Drop (clear) clipboard Ctrl-D
Select events by channel Ctrl-E
Analyze notes for scale Ctrl-K
Select notes by channel Ctrl-N
Move selected notes by snap or note value Arrow-left, -right, -up, -down
Delete selected notes Delete or Backspace
Repitch selected notes c
Edge-fix notes f
Toggle recording o
Quantize notes timing q
Randomize select notes (amplitude) r
Tighten notes timing t
Remove unlinked notes u
Verify and link notes u
