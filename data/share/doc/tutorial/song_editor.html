Seq66 Basic Tutorial

Song Editor

 
The figure below shows the external (as opposed to the Song tab) Song Editor window. It is a bit small, so run qseq66 (qpseq66.exe in Windows) to see the window elements.
 
  • Top bar. These controls support undo/redo, follow-progress, zoom, snap, entering "triggers", transposition, and editing the layout. Also shows the time duration of the layout.
  • Time line. Provides for vertical zoom and the markers "L" and "R" for looping. The L/R markers can be moved; a starting position can be set.
  • Names panel. Shows much the same information as does a pattern slot. Shows the mute status. A track can be click here to toggle its mute status.
  • Piano roll. The piano roll shows the layout of the patterns in a song. It starts out empty, then one can paint layouts of each track. The layouts can snap to a given resolution, and can be split or deleted. The File / Export / Export song action can be used to save the song in a format that other sequencers can read.

This window is packed with functionality. Be sure to try clicking or right clicking on the grid and the various buttons. Hover the mouse cursor over an interface item; often a tool-tip will appear.
