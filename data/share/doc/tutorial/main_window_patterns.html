Pattern SLots

 

 

  • The Slot. Pattern slots are the square buttons in the Live Grid. Note the following contents of a slot, if it is not an empty/blank slot:
    • Track Title. This holds a fairly short title for the slot's track.
    • Measures. The number of measures a pattern covers can be any value. Generally, though, patterns are short: 1, 2, 4, 8 etc. measures. Whatever the song needs.
    • Status. Most often, a track is either "Muted" (silent) or "Armed". Other statuses are "Queued" and "One-shot", which are explained in the user's manual.
    • Progress Box. The rectangle in the center of the slot represents the "color" of the track (if any), the events in the track, and the progress bar.
    • Pattern Number. Each pattern has a number ranging from 0 to 31. The pattern numbers are organized so that the row varies fastest; this can be changed by a setting as noted in the user's manual.
    • Port-Channel Numbers. Each pattern can be assigned to a given MIDI port and MIDI channel. Ports are tagged by a small integer starting at 0; the port names can be seen in the Edit / Preferences MIDI tabs. Channels range from 1 to 16, and control output channel of the events. "F" means that events are sent on whatever channel the event specifies.
    • Time Signature. This specifies the time signature of the pattern, which merely determines how the track looks in the pattern editor.
  • Right-click Menu. The right click menue provides for creating and editing a pattern, which can also be done by a double-click or by using a hot-key combination. Patterns can also be copied, moved, merged, and have a color specified.

Play with the slots and discover what they can do.

 

Remember that the pattern slots go by many names: tracks, loops, sequences (kind of misnomer, patterns, and slots.
