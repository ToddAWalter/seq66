This document describes the basic workings of Seq66, and includes enough to get started with creating a song and understanding the configuration. It is a quick introduction. For the complete manual, see the Help / User Manual command. If this fails, send an e-mail reporting this problem and your setup (operating system, default-app settings, etc.) to "ahlstromcj@gmail.com" as shown in Help / About, or look for the complete manual, seq66-user-manual.pdf, which can be found /usr/share/doc/seq66-0.98 or /usr/local/share/doc/seq66-0.98 on Linux, or in C:/Program Files (x86)/Seq66/doc on Windows. It can also be found in the Seq66 source code package in the seq66/data/share/doc directory.


This tutorial covers Seq66 basics useful to persons new to this live-grid sequencer application. It covers only the main window, the pattern-editor, the song-editor window, and a bit about configuration. The event-editor, mute-groups, sets, and session tabs, as well as the usage of MIDI controllers, are discussed in the rather extensive user's manual.

 

This tutorial covers Seq66 basics useful to persons new to this live-grid sequencer application. It is intended to get the reader up and running quickly.

It starts with a quick survey of the most important elements, which cover creating tracks (patterns) and modifying them. More advanced elements (sets, mute-groups, playlists) are best covered in the user's manual, not this tutorial.

Once the basics have been summarized, the tutorial begins, covering startup, configuration, creating a tune, and laying it out as a song performance.

 
 

As time goes on, this tutorial will be tweaked, especially as requested by user input at GitHub: ahlstromcj/seq66
