Seq66 Basic Tutorial

Main Window

 

 

  • Menu. Provides some expected functions, including opening, saving, import/export, recent files, help....
  • Main Top bar. Provides status and time information, and allows for changing the MIDI bus and time signature of the whole song.
  • Tabs. The tabs provide compact access to various views of the song.
  • Patterns Panel (Live Grid). This item show a single set of patterns. Patterns are also called "tracks", "loops", or "sequences". The patterns can be muted/unmuted by clicking on them. Each pattern can also be opened for editing via a right-click menu. Keystrokes ("hot-keys") can also activate or edit a pattern.
  • Grid Mode. This control selects what a click or hot-key does to a pattern. The default ("Loop") toggles the track. "Record" mode toggles recording. There are other modes as well.
  • Main Bottom. The top row at the bottom allows for naming and change the active set. The bottom row are standard play/record buttons, plus a Song/Live toggle button and a beats-per-minute indicator.

This window is packed with functionality. Be sure to try clicking or right clicking on the grid and the various buttons. Hover the mouse cursor over an interface item; often a tool-tip will appear.
