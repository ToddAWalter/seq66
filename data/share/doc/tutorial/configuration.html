Seq66 Basic Tutorial

Configuration

 
The figure below shows the Edit / Preferences window. It is a bit small, so run qseq66 (qpseq66.exe in Windows) to see the window elements. It provides a way to configure and restart Seq66, though it does not handle all configuration items.
Seq66 Configuration. The configuration is stored in a number of files stored in the user's configuration folder. Linux: /home/user/.config/seq66. Windows: C:/Users/user/AppData/Local/seq66/. Some settings can be made in Edit / Preferences; other less-used options require editing the file. The file extensions:
  • rc. This file (e.g. qseq66.rc) is the main configuration file. It specifies the MIDI I/O ports, port mapping, names and status of the other configuration files, a list of "recent files", and other settings.
  • usr. This file configures options instrument and buss name overrides, many user-interface settings, some default song settings, and a few more.
  • ctrl. Provides keystrokes and MIDI events to control Seq66. By default, only keystrokes are provided. Edited this file to add MIDI control events, MIDI status-display events, and MIDI macros. A Novation LaunchPad Mini configuration is provided as an example.
  • mutes. This file contains mute-group settings. These can be edited in the Live grid and in the Mutes tab. See the user manual.
  • playlist. This file contains play-lists, editable in the Playlist tab or in the play-list file. See the user manual.
  • drums. This file contains note remappings, useful for converting from old MIDI drum sets to GM drum sets. Also called the "note mapper". See the user manual.
  • palette. This file allows control of pattern colors, user-interface coloring in the Seq66-drawn areas, such as piano rolls. It can change brush patterns for drawing notes, scales, and background patterns. It must be edited by hand.

  • qss. These files are Qt style sheets, which can be used to change the colors of Qt-drawn elements, overriding the theme settings

This editor is jam-packed with functionality. Be sure to try clicking or right clicking on the grid and the various buttons. Hover the mouse cursor over an interface item; often a tool-tip will appear.
